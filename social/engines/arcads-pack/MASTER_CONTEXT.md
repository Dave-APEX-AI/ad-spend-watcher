# MASTER_CONTEXT — CaillteAI (Arcads pack)

Workspace memory for the Arcads engine when producing CaillteAI creative. Read this
first, then `../../BRAND.md` for the full brand bible. **No API keys in this file, ever.**

## Brand
- **CaillteAI** — AI voice receptionist + automations + smart websites for UK & Irish
  trades. *Caillte* (Irish) = "lost"; the promise is "never lose another call/lead/job".
- Voice: plain, confident, cheeky; site-foreman not SaaS; numbers over adjectives; trade
  idiom; ≤1 emoji; never hype/guru/jargon. Full detail in `../../BRAND.md`.

## Visual defaults
- Palette: ink #0A0F0D, emerald #0FB97E, cream #F5F3EC, amber #FFC247 (= loss/missed).
- 1080×1350 feed / 1080×1920 reels. Burned-in captions. Green-dot / amber-dot / soundwave
  motif. Faceless brand — AI actor plays the "everyman trade owner", never a real customer.
- End card: green soundwave + "Don't let leads go *caillte*." + free missed-call audit CTA.

## Default creative brief
- Scenario: a missed-call moment (emergency leak at 9pm, customer ringing round three
  trades) → the cost (a real £ number) → CaillteAI answers/books → calm payoff.
- Scripts: `../../series/office-heroes.md` (formats 1/2/5). Hooks: `../../content/hooks.md`.

## Honesty
AI-actor testimonials are dramatisations — label them. Never imply a real endorsement.

## Publishing
`meta-ad-builder` creates **paused** Meta ads only. Dave launches manually.

## Cost log

before spending.
